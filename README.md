# 简介

👯✨😄📫

前端的帧动画制作器,作为用户的你可以直接绘制多个帧，然后点击导出(import),就可以获得相应的css代码!

css代码可以直接置入你的前端项目中使用！✌️

开源不易，如果你觉得有用，就给个star吧！😋

# 功能

* [x] 绘笔功能

* [x] 帧时间轴管理

* [x] 图片插入

# 如何使用？

## 1.无网络下本地运行

1.克隆项目```git clone git@github.com:a1163675107/easy-animation.git```

2.进入项目根目录,启动指令:```npm install```

3.```npm run serve```
